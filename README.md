# K15 FM - Thanh xuân phát lại

Website tĩnh chạy trên GitHub Pages.

## Cấu trúc file
- `index.html`: Trang chính / thư mời
- `style.css`: Giao diện
- `script.js`: Countdown và hiệu ứng
- `CNAME`: Tên miền `k15hvnh.com`
- `.nojekyll`: Giúp GitHub Pages publish website tĩnh ổn định
- `assets/images/hoc-vien-ngan-hang.jpg`: Ảnh nền

## Cách cập nhật GitHub
Upload toàn bộ các file/thư mục này vào branch `main`, thư mục gốc `/root`.
