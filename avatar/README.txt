K15FM Avatar Studio V4

Tính năng:
- 3 chế độ: Classic, Gold Premium, FM Retro.
- Dùng đúng frame designer đã chốt làm Classic.
- Chuyển frame không cần upload lại ảnh.
- Ảnh khớp đúng vùng trong suốt của frame, không cắt lệch vòng.
- Xuất avatar PNG HD 1080x1080.

Cách cập nhật GitHub Pages:
1. Giải nén file zip.
2. Upload đè nguyên thư mục `avatar` lên repo k15hvnh.
3. Đợi GitHub Pages deploy.
4. Mở:
   https://k15hvnh.com/avatar/?v=v4

Nếu vẫn thấy bản cũ: Ctrl + F5 hoặc mở tab ẩn danh.
