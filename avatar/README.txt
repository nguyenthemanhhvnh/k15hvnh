K15FM Avatar Studio V3.1

Bản này đã sửa lỗi lộ vệt trắng/tam giác trắng ở vùng avatar:
- Ảnh được clip theo đúng vòng tròn avatar.
- Canvas có nền navy bên dưới.
- Thuật toán fit ảnh theo đường kính vòng avatar, không theo toàn canvas.

Cách cập nhật trên GitHub:
1. Giải nén file zip.
2. Upload đè toàn bộ file trong thư mục avatar hiện tại:
   - index.html
   - css/style.css
   - js/app.js
   - images/*
3. Đợi GitHub Pages deploy.
4. Mở lại: https://k15hvnh.com/avatar/

Nếu trình duyệt vẫn hiện bản cũ, nhấn Ctrl + F5.
