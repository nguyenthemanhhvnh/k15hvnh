K15FM Avatar Studio FINAL V4

Cách cập nhật:
1. Giải nén zip.
2. Upload đè nguyên thư mục avatar lên GitHub repo k15hvnh.
3. Đợi GitHub Pages deploy.
4. Mở https://k15hvnh.com/avatar/?v=final-v4
5. Nếu vẫn bị cache, nhấn Ctrl+F5 hoặc mở tab ẩn danh.

Điểm khác biệt bản Final:
- Làm sạch lại frame PNG bằng alpha thật.
- Xóa triệt để nền caro/trắng trong vùng avatar.
- Code render ảnh theo vùng tròn avatar, không theo toàn canvas.
- Ảnh luôn phủ kín vòng tròn, không lộ vệt trắng khi xoay/zoom.
