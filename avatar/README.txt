# K15FM Avatar Studio

## Cách dùng nhanh

1. Giải nén file `k15fm-avatar-studio.zip`.
2. Upload toàn bộ thư mục lên hosting của website.
3. Mở đường dẫn dạng:

   https://tenmiencuaban.vn/avatar/

4. Người dùng chỉ cần chọn ảnh, kéo/zoom/xoay rồi bấm "Tải avatar HD".

## Cấu trúc file

- `index.html`: giao diện chính
- `style.css`: giao diện navy - gold
- `script.js`: xử lý upload, kéo ảnh, zoom, xoay, tải ảnh
- `frame.png`: khung avatar K15FM

## Muốn thay frame

Chỉ cần thay file `frame.png` bằng file frame mới, giữ nguyên tên `frame.png`.

Lưu ý: file frame nên là PNG 1080x1080 và vùng đặt ảnh cần trong suốt.
