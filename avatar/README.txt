K15FM Avatar Studio - Final Corrected

Bản này sửa lỗi cắt sai vòng tròn.

Điểm khác biệt:
- Không dùng clip tròn thủ công nữa.
- Dùng đúng vùng trong suốt có sẵn của frame designer.
- Frame designer giữ nguyên chi tiết.
- Ảnh người dùng nằm sau frame, tự hiện qua vùng trong suốt.
- Không còn lệch vòng tròn.

Cách cập nhật:
1. Giải nén file zip.
2. Upload đè nguyên thư mục `avatar` lên repo k15hvnh.
3. Đợi GitHub Pages deploy.
4. Mở:
   https://k15hvnh.com/avatar/?v=final-corrected

Nếu vẫn thấy bản cũ: Ctrl + F5 hoặc mở tab ẩn danh.
